Быстрый старт

Шаг 1

Включите Отладку по USB на вашем устройстве:

  • Настройки → Об устройстве → ПО → Жмите (7 раз) на НОМЕР СБОРКИ, пока не станете Разработчиком!
  • Настройки → Для разработчиков → Отладка по USB → Включить → Готово!

Шаг 2

Подключите ваше устройство к компьютеру при помощи USB-кабеля.

Дайте разрешение на отладку на экране вашего устройства. Не забудьте выбрать опцию "Всегда Разрешать".

Шаг 3

Если вы подключены по USB, нажмите на кнопку "Отладка по сети" (вверху справа) и вы подключитесь с помощью Wi-fi автоматически.

Либо узнайте IP вашего Android-устройства вручную: Настройки → Об устройстве → Состояние → IP Адрес.

Введите IP в поле программы и нажмите "Подключиться".
