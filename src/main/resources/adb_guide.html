Команды ADB

ADB (Android Debug Bridge) — это мощный инструмент для взаимодействия с Android-устройством.

Базовые команды

  • adb devices
    — Показывает список подключенных устройств.
  • adb shell
    — Открывает терминал устройства.
  • adb install app.apk
    — Устанавливает приложение.
  • adb logcat
    — Просмотр логов системы.

Работа с файлами

  • adb pull /sdcard/file.txt
    — Копирует файл с устройства.
  • adb push file.txt /sdcard/
    — Загружает файл на устройство.

Wi-Fi подключение

  1. Подключите устройство по USB.
  2. Выполните adb tcpip 5555.
  3. Отключите USB-кабель.
  4. Запустите adb connect :5555.
